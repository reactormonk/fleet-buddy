# fleet-buddy
Eve Online fleet buddy based on the CREST API

# Install
Rename `project/secrets.scala.example` and change it as needed.
Same with `server/src/main/resources/secrets.conf.example`

# semantic ui setup

```bash
cd client
npm install semantic-ui --save
cd semantic
gulp build
```

# css setup

```bash
cd client
npm install elm-css
```
